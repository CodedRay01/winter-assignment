Attributes of the Table in css

Attribute Description
border Specifies the width of the border. Default value is 0.
border-collapse Specifies whether the table borders should be collapsed into a single border or not. Default value is separate.
border-spacing Specifies the distance between the borders of adjacent cells. Default value is 2px.
caption-side Specifies the placement of a table caption. Default value is top.
empty-cells Specifies whether or not to display borders and background on empty cells in a table. Default value is show.
table-layout Specifies the algorithm used to lay out the table cells, rows, and columns. Default value is auto.
width Specifies the width of the table. Default value is auto.
colspan Specifies the number of columns a cell should span. Default value is 1.
rowspan Specifies the number of rows a cell should span. Default value is 1.
background-color Specifies the background color of a table. Default value is transparent.
color Specifies the color of the text in a table. Default value is depends on the browser.
text-align Specifies the horizontal alignment of the text in a table. Default value is left.
vertical-align Specifies the vertical alignment of the text in a table. Default value is middle.
padding Specifies the padding of a table. Default value is 1px.
hover Specifies the hover effect on the table. Default value is none.
border-radius Specifies the border radius of the table. Default value is 0px.

1. border

The border attribute specifies the width of the border around a table. The default value is 0.

table,td,th{
border: 1px solid black;
}

Row 1, Cell 1 Row 1, Cell 2
Row 2, Cell 1 Row 2, Cell 2

2. border-collapse

The border-collapse attribute specifies whether the table borders should be collapsed into a single border or not. The default value is separate.
